实现字符串查找函数 indexOf

用例 1:

```js
const str = 'hello world';

indexOf(str, 'l'); // 2
```

用例 2:

```js
const str =
  'There is magic in the trying and learning and trying again.. but any unprocessed PTSD will come back to haunt you';

indexOf(str, 'PTSD'); // 81
```

用例 3:

```js
const str = 'hello world';

indexOf(str, 'ol'); // -1
```
